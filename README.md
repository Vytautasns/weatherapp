# Sample Weather Page

Angular 1.x weather app using openWeatherMap api.

### Prerequisites

Need to have npm installed.

### Installing


Clone the repository

```bash
git clone
```

Intall dependencies

```bash
cd weatherApp
npm install
```

Start local server

```bash
npm start
```

Visit page on http://127.0.0.1:8080/
